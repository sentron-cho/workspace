디스플레이(display) 속성

SPAN 태그는 기본이 인라인이다

P 태그는 기본이 블록이다

P 태그를 인라인으로(width, height 속성 무시)

SPAN 태그를 블록으로

P 태그를 (display: inline-block; width: 300px; height: 80px;) 속성으로

플로팅(float) 속성

left 속성 1 left 속성 2 right 속성 1 right 속성 2
inherit(부모 right) 속성

포지셔닝(position) 속성

relative left 100px top 10px relative left 20px bottom 10px absolute left 10px bottom 10px absolute right 10px bottom 10px
